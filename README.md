# UberLogin
Uber Login SDK - light version

## How to install?
### Step 1:
Add it in your root build.gradle at the end of repositories:

```
  allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

### Step 2:
Step 2. Add the dependency

```
	dependencies {
	        compile 'com.github.natiginfo:UberLogin:v1.0'
	}
```
